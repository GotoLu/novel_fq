# 过审自查清单

## 使用说明

本文档为创作过程中的过审自查清单，建议在以下时机使用：
- 每章完成后
- 关键情节创作前
- 敏感内容创作时
- 完稿后全面自查

---

## 一、朝代表述自查

### 1.1 必须检查

- [ ] **是否使用了真实历史朝代名称？**
  - 禁止：明朝、明代、明国、唐代、宋朝、清朝等
  - 允许：天启朝、天启年间、本朝、当今圣上、朝廷

- [ ] **是否使用了"天启朝""天启年间"表述？**
  - 必须使用虚构朝代名称
  - 年号必须为"天启"

- [ ] **是否涉及真实历史事件？**
  - 禁止涉及任何真实历史事件
  - 所有历史事件必须虚构

- [ ] **是否对真实历史人物进行了描写？**
  - 禁止对真实历史人物进行任何描写
  - 所有历史人物必须虚构

- [ ] **是否有历史虚无主义倾向？**
  - 禁止否定历史规律
  - 禁止篡改真实历史

### 1.2 正确表述示例

✅ 正确：
- "天启朝中后期"
- "天启二十三年"
- "本朝开国皇帝"
- "当今圣上沉迷修道"

❌ 错误：
- "明朝中后期"
- "嘉靖年间"
- "明太祖"
- "朱元璋"

## 二、能力体系自查

### 2.1 必须检查

- [ ] **是否使用了"迷信""算命""占卜"等敏感词汇？**
  - 禁止：迷信、算命、占卜、风水、符咒、法术
  - 允许：血脉觉醒、远古能力、命运感知、望气术

- [ ] **是否使用了"血脉觉醒""命运感知"等玄幻表述？**
  - 必须使用玄幻化表述
  - 明确定义为"玄幻设定"

- [ ] **能力是否有代价和限制？**
  - 每个能力必须有代价
  - 每个能力必须有限制
  - 避免无代价的超能力

- [ ] **能力原理是否有合理解释（基因、血脉、精神力）？**
  - 用科学化、系统化的方式解释
  - 用基因、血脉、精神力等概念

- [ ] **是否过度强调超能力？**
  - 避免能力过于强大
  - 强调智慧、策略、成长

- [ ] **能力等级是否清晰？**
  - 能力等级必须清晰
  - 成长必须合理
  - 逻辑必须自洽

### 2.2 正确表述示例

✅ 正确：
- "血脉觉醒"
- "远古能力"
- "命运感知"
- "望气术"
- "精神力消耗"

❌ 错误：
- "算命"
- "占卜"
- "迷信"
- "符咒"
- "法术"

## 三、内容自查

### 3.1 必须检查

- [ ] **暴力描写是否过度？**
  - 控制暴力描写程度
  - 避免过度血腥
  - 重点描写策略和智慧

- [ ] **是否有血腥恐怖内容？**
  - 避免过度血腥
  - 避免过度恐怖
  - 避免详细描写暴力过程

- [ ] **是否有政治隐喻？**
  - 避免对政治制度进行批判
  - 避免影射现实政治

- [ ] **是否影射现实政治？**
  - 禁止影射任何现实政治
  - 禁止对现实政治进行批判

- [ ] **内容是否健康？**
  - 保持内容健康
  - 避免低俗内容
  - 避免色情内容

### 3.2 内容把控要点

**暴力描写**：
- 用"战斗""对决""制服"等词汇
- 重点描写策略、智慧、成长
- 避免详细描写血腥场面

**政治内容**：
- 强调个人成长和命运
- 避免政治斗争描写
- 东厂设定为玄幻组织

## 四、价值观自查

### 4.1 必须检查

- [ ] **主角选择是否极端？**
  - 主角最终选择平衡
  - 避免极端选择

- [ ] **是否对文明社会进行否定？**
  - 不否定文明的价值
  - 只是批判过度压制
  - 强调平衡和选择

- [ ] **是否有负面价值导向？**
  - 保持积极向上的价值导向
  - 强调正能量

- [ ] **主题是否积极向上？**
  - 主题是"寻找真正的自我"
  - 强调成长、选择、责任

- [ ] **是否强调个人成长和责任？**
  - 强调个人成长
  - 强调个人选择
  - 强调个人责任

### 4.2 价值观把控要点

**核心主题**：
- "寻找真正的自我"
- "平衡文明与天性"
- "个人成长和选择"

**避免极端**：
- 不完全脱离社会
- 不完全被驯化
- 选择平衡之路

## 七、常见问题与解决方案

### 7.1 朝代表述问题

**问题**：不小心使用了"明朝"字眼
**解决**：全文搜索"明朝"，替换为"天启朝"或"本朝"

**问题**：使用了真实历史年号
**解决**：全文搜索真实年号，替换为"天启"

### 7.2 能力表述问题

**问题**：使用了"算命"等敏感词汇
**解决**：替换为"望气术""命运感知"等玄幻表述

**问题**：能力过于强大，无代价
**解决**：增加能力代价和限制，强调精神力消耗

### 7.3 内容问题

**问题**：暴力描写过度
**解决**：减少血腥描写，重点描写策略和智慧

**问题**：有政治隐喻
**解决**：删除政治隐喻内容，强调个人成长

### 7.4 价值观问题

**问题**：主角选择极端
**解决**：修改为主角选择平衡，强调成长和责任

**问题**：否定文明社会
**解决**：修改为批判过度压制，而非否定文明
