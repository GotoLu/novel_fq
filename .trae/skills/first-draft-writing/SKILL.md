# 初稿写作技能

## 核心能力描述

初稿写作是将构思转化为文字的核心创作阶段，该技能帮助创作者按照大纲逐章写作、保持创作节奏、克服写作障碍，完成小说的初稿创作。

## 关键操作步骤

### 1. 写作准备
- **工作环境**：
  - 安静舒适的写作空间
  - 固定的写作时间
  - 减少干扰和打断
- **工具准备**：
  - 写作软件（Word、Scrivener、Typora等）
  - 大纲和人物小传随时查阅
  - 备份机制（云同步、版本控制）
- **心理准备**：
  - 接受初稿不完美的现实
  - 建立完成优先于完美的信念
  - 克服完美主义倾向

### 2. 写作节奏管理
- **日常计划**：
  - 设定每日写作目标（字数或时间）
  - 建议每日2000-5000字或2-4小时
  - 固定写作时段，养成习惯
- **周期计划**：
  - 每周写作计划
  - 每月进度目标
  - 预计完成时间
- **进度跟踪**：
  - 记录每日完成字数
  - 可视化进度条
  - 适时调整计划
- **弹性调整**：
  - 预留缓冲时间
  - 根据实际调整目标
  - 避免过度压力

### 3. 章节写作流程
- **写作前**：
  - 回顾本章大纲
  - 回顾前文衔接
  - 明确本章目标
- **写作中**：
  - 快速推进，不过度修改
  - 遇到问题标记，继续前进
  - 保持叙事流畅
  - 注意章节结尾钩子
- **写作后**：
  - 简单通读检查
  - 记录本章要点
  - 更新进度
  - 简短休息

### 4. 场景写作技巧
- **场景构成**：
  - 时间和地点
  - 参与人物
  - 核心事件
  - 情感基调
- **场景展开**：
  - 从具体细节切入
  - 通过人物视角展开
  - 对话、行动、心理交织
  - 推进情节或揭示人物
- **场景转换**：
  - 明确的转场标志
  - 时间/空间的跳跃
  - 视角的切换
  - 情绪的转换

### 5. 对话写作技巧
- **对话功能**：
  - 推进情节
  - 揭示人物性格
  - 展现人物关系
  - 提供信息
- **对话原则**：
  - 符合人物身份和性格
  - 有潜台词和言外之意
  - 避免直白说教
  - 有来有往，有冲突张力
- **对话格式**：
  - 对话提示语的运用
  - 动作和神态的配合
  - 节奏和停顿的控制

### 6. 描写技巧
- **人物描写**：
  - 外貌描写抓特征
  - 动作描写显性格
  - 心理描写挖深度
  - 语言描写见个性
- **环境描写**：
  - 渲染氛围
  - 推动情节
  - 象征暗示
  - 控制篇幅，避免冗长
- **细节描写**：
  - 选择有意义的细节
  - 调动五感
  - 具体生动
  - 以小见大

### 7. 叙事技巧
- **叙事视角**：
  - 第一人称（代入感强）
  - 第三人称限制视角（灵活）
  - 第三人称全知视角（宏大）
  - 多视角切换
- **叙事节奏**：
  - 动作场景快节奏
  - 情感场景慢节奏
  - 张弛有度
  - 控制信息密度
- **叙事手法**：
  - 顺叙、倒叙、插叙
  - 伏笔和呼应
  - 悬念和揭秘
  - 对比和衬托

### 8. 写作障碍克服
- **卡文应对**：
  - 暂时跳过，标记待补
  - 回顾大纲，找回方向
  - 自由写作，打破僵局
  - 改变场景或人物
- **拖延应对**：
  - 设定小目标，降低门槛
  - 建立写作仪式感
  - 找写作伙伴互相督促
  - 记录拖延原因，针对性解决
- **自我怀疑**：
  - 接受初稿的粗糙
  - 相信修改的力量
  - 关注进度而非质量
  - 适度寻求鼓励

### 9. 写作日志记录
- **每日记录**：
  - 完成字数和时间
  - 写作时的状态和感受
  - 遇到的问题和解决方法
  - 明日计划
- **定期总结**：
  - 每周进度总结
  - 阶段性回顾
  - 调整写作策略
  - 庆祝里程碑

## 评估标准

### 定量指标
- **进度达成率**：实际字数/计划字数 ≥ 90%
- **写作持续性**：每周写作天数 ≥ 5天
- **初稿完成度**：按计划完成初稿
- **字数达标**：达到预定总字数

### 定性指标
- **情节推进**：每章都有实质进展
- **人物塑造**：人物行为符合设定
- **叙事流畅**：故事讲述清晰流畅
- **完成优先**：不过度纠结细节，持续推进
- **节奏稳定**：保持稳定的写作节奏

### 能力等级
- **初级**：能完成初稿，但进度不稳定，质量起伏大
- **中级**：能按计划完成初稿，节奏稳定，质量均衡
- **高级**：初稿质量较高，节奏完美，能灵活应对障碍
- **专家级**：初稿接近定稿质量，创作高效流畅，风格鲜明

## 实践建议

1. **建立仪式**：固定的写作时间和环境
2. **小步快跑**：设定小目标，持续前进
3. **允许粗糙**：初稿重在完成，不在完美
4. **及时记录**：记录灵感和问题，不中断写作流
5. **适度休息**：避免疲劳创作，保持精力
6. **寻求支持**：写作社群或伙伴互相鼓励

- 问题标记清单（待修改的问题）
- 创作心得总结

## 约束条件：章节写作自检

### 强制要求

**每章写作完成后，必须进行以下自检，确保质量符合标准。**

### 章节写作核心约束（15项）

#### 1. 叙事视角一致性
- 章节内采用的叙事视角（第一人称、第三人称限制视角或全知视角）必须保持连贯
- 避免视角混乱或无理由切换
- 视角切换必须有明确的标志和叙事需要

#### 2. 场景构建与氛围营造
- 必须通过环境描写、感官细节（视觉、听觉、嗅觉、触觉、味觉）构建清晰的场景画面
- 必须营造符合情节需求的氛围
- 场景描写必须服务于情节和人物

#### 3. 情节推进与节奏控制
- 章节内容必须有效推动主线或支线情节发展
- 必须合理安排叙事节奏
- 必须平衡动作场景与心理描写、对话与叙述的比例

#### 4. 人物塑造深化
- 必须通过人物的行为、心理活动、环境互动和反应，展现人物性格特点
- 必须展现人物内心变化或背景故事
- 人物行为必须符合其设定逻辑

#### 5. 背景信息处理
- 必须合理穿插必要的背景信息或世界观设定
- 避免信息堆砌或解释不足
- 必须确保读者能理解情节发展的上下文

#### 6. 冲突设置与张力维持
- 章节内必须包含适当的内部冲突（人物内心矛盾）或外部冲突（人物间矛盾、环境挑战等）
- 必须维持读者阅读兴趣和情节张力

#### 7. 章节结构完整性
- 章节必须有明确的开端、发展和结尾
- 必须形成相对完整的叙事单元
- 必须为后续章节留下适当的悬念或衔接点

#### 8. 语言风格与描写技巧
- 必须保持语言风格的统一性
- 必须运用多样化的描写技巧
- 避免重复单调的表述

#### 9. 细节逻辑合理性
- 必须检查时间线、因果关系、人物能力与环境限制等细节的逻辑一致性
- 避免出现情节漏洞或设定矛盾

#### 10. 主题与象征元素
- 章节内容必须服务于小说的整体主题
- 适当运用象征元素、隐喻或伏笔
- 增强作品的深度和层次感

#### 11. 伏笔与呼应
- 必须检查是否埋下伏笔或呼应前文伏笔
- 确保情节连贯性
- 伏笔必须自然，不刻意

#### 12. 读者体验
- 必须考虑读者的阅读体验
- 必须设置适当的悬念
- 必须引发读者情感共鸣

#### 13. 文体风格一致性
- 文体风格必须与整部小说保持一致
- 避免风格跳跃
- 必须符合小说的整体基调

#### 14. 对话功能明确性
- 对话必须推动情节、展现人物、揭示信息
- 避免无效对话

#### 15. 章节独立性
- 章节必须相对独立
- 读者必须能理解本章内容
- 必须与前后章节衔接自然

### 对话专项约束

#### 对话自然性
- 对话必须符合人物身份和性格
- 对话必须符合人物的语言风格
- 对话必须自然流畅，不生硬
- 对话必须有潜台词和言外之意

#### 对话逻辑性
- 对话必须有明确的逻辑顺序
- 对话必须推动情节发展
- 对话必须展现人物关系
- 对话必须避免重复和冗余

#### 对话符合人物设定
- 对话必须符合人物的性格特点
- 对话必须符合人物的教育背景
- 对话必须符合人物的身份地位
- 对话必须符合人物当前的情绪状态

### 自检流程

#### 第一步：快速通读
- 通读全章，检查流畅性
- 标注明显问题
- 不做详细修改

#### 第二步：逐项检查
- 对照15项核心约束逐项检查
- 记录所有不符合项
- 标注需要修改的地方

#### 第三步：专项检查
- 对话专项检查
- 时间线检查
- 人物一致性检查
- 设定一致性检查
- 伏笔检查

#### 第四步：问题汇总
- 汇总所有问题
- 按优先级排序
- 制定修改计划

#### 第五步：记录存档
- 记录自检结果
- 存档自检清单
- 为后续修改提供参考

### 自检记录要求

**每章必须记录以下内容**：

1. **章节信息**：
   - 章节编号和标题
   - 自检日期
   - 自检人

2. **自检结果**：
   - 核心要点符合项数/15
   - 对话检查符合项数/12
   - 专项检查符合项数/20

3. **发现问题**：
   - 问题描述
   - 问题位置
   - 严重程度（高/中/低）
   - 修改建议

4. **修改计划**：
   - 优先级排序
   - 修改方案

5. **自检结论**：
   - 通过/需修改/需重大修改

### 语言风格参考（强制）

**写作时必须参考人物语言风格表**：
1. 每个主要人物的对话必须符合其语言风格设定
2. 对话必须体现人物当前成长阶段的语言特征
3. 如发现语言风格表不够详细，应回溯补充而非自行发挥
4. 语言风格表路径：`/Users/luxd/workspace/testCode/novo-test/.trae/novel/characters/character-language-styles.md`

### 过审约束遵守

**初稿写作必须遵守市场调研阶段确定的过审约束**：
1. 朝代表述：必须使用架空/虚构朝代名称，禁止真实历史朝代名称
2. 能力表述：必须使用玄幻表述，禁止迷信敏感词汇
3. 历史处理：完全架空，不涉及真实历史事件
4. 价值观：积极向上，强调成长和责任
5. 内容把控：避免过度暴力、血腥、政治隐喻

### 变更传播机制

**初稿写作阶段发现设定问题需要修改时，必须触发变更传播检查**：

1. **识别变更影响范围**：修改涉及哪些设定？
2. **列出受影响文档**：大纲、世界观、人物小传等是否需要同步更新？
3. **逐一更新**：按清单更新所有受影响的文档
4. **验证一致性**：确认更新后的跨阶段一致性

**变更分级**：
- **S级变更**（核心设定变更）：全链路更新，如新增能力机制、核心情节改写
- **A级变更**（重要设定调整）：更新直接关联文档，如人物行为逻辑调整
- **B级变更**（细节修改）：只更新当前文档，如对话措辞修改

**影响范围识别清单**（按层级排查）：
- 第一层：直接关联（当前章节 → 前后章节 → 伏笔呼应）
- 第二层：人物关联（主角小传 → 反派小传 → 配角小传 → 人物关系 → 语言风格）
- 第三层：世界观关联（能力体系 → 社会规则 → 历史设定）
- 第四层：大纲关联（章节大纲 → 冲突地图 → 关键场景 → 伏笔设计）

**更新顺序规则**：
1. 先更新世界观设定（底层设定）
2. 再更新人物设定
3. 再更新大纲
4. 最后更新写作指南和当前章节

**变更记录模板**：
```
变更编号：CP-FD-[序号]
变更日期：YYYY-MM-DD
触发阶段：初稿写作
变更内容：[具体描述]
变更原因：[为什么需要变更]
变更级别：S/A/B级
受影响文档：[列表]
更新状态：进行中/已完成
验证结果：通过/未通过
```

**变更回滚机制**：
- 变更前备份当前版本
- 变更过程中如发现方向不对，可回滚到变更前版本
- 变更完成后验证不通过，也可回滚

**关系链断裂检测**（S/A级变更必须执行）：
1. 新设定是否与现有角色身份冲突？（每个角色的身份/职业/知识是否与新设定矛盾）
2. 新设定是否与现有角色关系冲突？（角色之间的信息差、互动是否与新设定矛盾）
3. 新设定是否与现有情节冲突？（已发生的情节、角色的行为是否与新设定矛盾）

**修复方案评估**（发现断裂点后）：
1. 逻辑自洽性：修复方案是否与现有设定逻辑自洽？
2. 戏剧张力：修复方案是否增加了戏剧张力？
3. 角色深度：修复方案是否深化了角色？
4. 过审风险：修复方案是否增加过审风险？
5. 修改成本：修复方案需要修改多少文档？

**断裂点依赖分析**（多个断裂点时）：
1. 列出所有断裂点
2. 分析断裂点之间的依赖关系
3. 确定修复顺序（先修复上游断裂点）
4. 上游修复方案确定后，再评估下游断裂点

**更新顺序策略选择**：
- 顺向传播（世界观→人物→大纲→写作）：变更源头在上游，按当前规则执行
- 逆向传播（初稿→大纲→人物→世界观）：变更源头在下游，需先评估底层能否支撑变更
  - 底层可以支撑：按当前规则执行
  - 底层无法支撑：先修复底层，再更新当前阶段

**变更可行性评估**（S级变更必须执行，在更新之前）：
1. 变更是否与核心主题冲突？（如果冲突，是否可以调和？）
2. 变更是否需要重写超过50%的文档？（如果是，考虑是否值得）
3. 变更是否改变故事的根本性质？（如果是，需要重新评估）
4. 变更的修复成本是否远大于收益？（如果是，考虑替代方案）
评估结果：可行 / 有条件可行 / 不可行（否决变更，寻找替代方案）

**回溯修改规则**（变更需要修改已完成阶段的文档时）：
1. 标记回溯原因和范围
2. 修改完成后重新执行该阶段的门控检查
3. 确认修改不会导致该阶段其他文档不一致
4. 只修改受影响的文档，不做"顺便"修改
5. 修改后必须更新变更记录
6. 从回溯阶段开始，逐阶段检查是否有新的不一致

**常见触发场景**：
- 写作中发现人物行为与设定矛盾，需要调整设定
- 写作中发现情节逻辑漏洞，需要调整大纲
- 写作中发现世界观设定不足，需要补充设定

### 阶段门控

**初稿写作完成后，必须通过以下门控检查才能进入下一阶段**：

1. **完整性检查**：初稿是否按大纲完成？字数是否达标？
2. **人物检查**：人物行为是否全部符合设定？对话是否符合语言风格？
