# 大纲规划技能

## 核心能力描述

大纲规划是小说创作的蓝图设计阶段，该技能帮助创作者构建完整的故事结构、规划情节发展、设计节奏控制，为后续写作提供清晰的路线图。

## 关键操作步骤

### 1. 主题确定
- **核心主题提炼**：
  - 从灵感中提炼核心思想
  - 确定主题的普世性和深度
  - 主题的多层次表达（表层/深层）
- **主题呈现方式**：
  - 显性表达 vs 隐性渗透
  - 通过人物命运体现
  - 通过情节冲突展现
  - 通过象征意象暗示
- **主题与时代**：主题的时代意义和现实关照

### 2. 核心冲突设计
- **冲突类型选择**：
  - 人与自然（生存挑战）
  - 人与社会（制度、观念、利益冲突）
  - 人与人（情感、利益、价值观冲突）
  - 人与自我（内心挣扎、成长蜕变）
- **冲突层次**：
  - 外部冲突（表层情节推动）
  - 内部冲突（人物内心矛盾）
  - 价值观冲突（深层思想碰撞）
- **冲突升级路径**：
  - 初始冲突设定
  - 冲突逐步升级
  - 冲突高潮设计
  - 冲突解决方式

### 3. 故事结构设计
- **三幕式结构**：
  - 第一幕：铺垫（占20-25%）
    - 开场钩子
    - 人物介绍
    - 世界观呈现
    - 激励事件
  - 第二幕：对抗（占50-60%）
    - 冲突发展
    - 情节转折
    - 中点高潮
    - 危机加深
  - 第三幕：解决（占20-25%）
    - 最终高潮
    - 冲突解决
    - 人物成长
    - 结局收尾

- **其他结构模式**：
  - 五幕式结构（铺垫-发展-高潮-回落-结局）
  - 多线叙事结构
  - 倒叙/插叙结构
  - 环形结构
  - 碎片化结构

### 4. 章节大纲制定
- **章节数量规划**：
  - 根据总字数确定章节数
  - 每章字数建议（3000-8000字）
  - 卷/部的划分
- **章节内容设计**：
  - 每章核心事件
  - 每章人物行动
  - 每章情感变化
  - 每章推进目标
- **章节节奏控制**：
  - 高潮章节分布
  - 过渡章节安排
  - 悬念设置节点
  - 信息释放节奏
- **章节钩子**：
  - 每章结尾钩子设计
  - 章节间衔接方式

### 5. 情节线索规划
- **主线情节**：
  - 主线发展脉络
  - 主线关键节点
  - 主线推进节奏
- **支线情节**：
  - 支线数量控制（2-5条）
  - 支线与主线关系
  - 支线交织节点
  - 支线收束方式
- **伏笔设计**：
  - 伏笔埋设位置
  - 伏笔回收节点
  - 伏笔隐蔽性控制

### 6. 字数与节奏规划
- **总字数确定**：
  - 根据题材和读者定位
  - 短篇：1-3万字
  - 中篇：3-10万字
  - 长篇：10-50万字
  - 超长篇/网文：100万字以上
- **节奏控制**：
  - 开篇节奏（快速吸引）
  - 发展节奏（张弛有度）
  - 高潮节奏（紧凑激烈）
  - 结尾节奏（从容收尾）
- **信息密度**：
  - 情节密度控制
  - 信息释放节奏
  - 留白与充实平衡

### 7. 关键场景设计
- **开场设计**：
  - 开场方式（悬念/冲突/氛围/人物）
  - 开场钩子强度
  - 开场信息量控制
- **高潮设计**：
  - 主要高潮场景（2-3个）
  - 高潮前铺垫
  - 高潮后缓冲
- **转折设计**：
  - 情节转折点（3-5个）
  - 转折合理性
  - 转折冲击力
- **结局设计**：
  - 结局类型（圆满/开放/悲剧/反转）
  - 结局合理性
  - 结局余味

- 叙事学和情节理论
- 戏剧冲突理论
- 类型小说创作规律

### 文学素养
- 中外经典作品结构分析
- 不同题材的结构特点
- 叙事节奏控制技巧
- 悬念和伏笔运用

### 逻辑思维
- 因果关系构建
- 情节合理性推演
- 人物行为逻辑
- 时间线管理

### 工具使用
- 大纲编写软件（Scrivener、yWriter等）

## 评估标准

### 定量指标
- **结构完整性**：包含开篇、发展、高潮、结局四要素
- **章节数合理**：总字数/章节平均字数 = 章节数
- **节奏分布**：高潮场景占比15-25%
- **伏笔回收率**：埋设伏笔回收率 ≥ 95%

### 定性指标
- **逻辑自洽**：情节发展符合因果逻辑
- **冲突强度**：核心冲突具有足够张力和深度
- **节奏合理**：张弛有度，无冗长或仓促感
- **主题清晰**：主题贯穿始终，表达明确
- **创新性**：结构有新意，不落俗套

### 能力等级
- **初级**：能制定基本大纲，但结构较简单
- **中级**：能设计完整结构，节奏控制合理
- **高级**：结构精巧，节奏完美，冲突设计深刻
- **专家级**：能创新结构模式，引领叙事革新

## 实践建议

1. **参考经典**：学习经典作品的结构设计
2. **反复推敲**：大纲需要多次修改和完善
3. **预留弹性**：大纲是指导而非束缚，保持调整空间
4. **验证逻辑**：检查情节发展的合理性和必然性
5. **节奏测试**：想象阅读体验，测试节奏是否舒适
6. **工具辅助**：使用大纲软件提高效率

## 输出成果

- 故事结构图（整体框架可视化）
- 章节大纲表（每章核心内容）
- 情节时间线（事件顺序和时长）
- 冲突发展图（冲突升级路径）
- 关键场景清单（重要场景设计）
- 节奏规划表（高潮和过渡分布）
- 伏笔设计表（埋设和回收计划）
