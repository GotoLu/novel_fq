# 定稿技能

## 核心能力描述

定稿是小说创作的收尾阶段，该技能帮助创作者完成最终校对、确定书名、准备出版材料、完成作者简介，确保作品以最佳状态进入出版流程。

## 关键操作步骤

### 1. 最终文字校对
- **全文通读**：
  - 最后一次完整通读
  - 以读者身份体验全文
  - 确认无遗漏问题
  - 检验整体阅读体验
- **重点检查**：
  - 开篇和结尾
  - 重要转折和高潮场景
  - 人物首次出场描写
  - 关键对话和细节
- **格式最终确认**：
  - 章节标题格式统一
  - 段落格式规范
  - 对话格式一致
  - 标点符号规范

### 2. 书名确定
- **书名原则**：
  - 简洁易记（2-6字为宜）
  - 符合题材和风格
  - 有吸引力和独特性
  - 无歧义和负面联想
- **书名类型**：
  - 人名型（《红楼梦》《哈利·波特》）
  - 事物型（《魔戒》《三体》）
  - 概念型（《活着》《百年孤独》）
  - 诗句型（《此间的少年》）
  - 地点型（《边城》）
- **书名测试**：
  - 朗朗上口，易于传播
  - 与同类作品书名对比
  - 征求他人意见
  - 检索是否重名
- **备选方案**：
  - 准备3-5个备选书名
  - 评估各书名的优劣
  - 根据反馈确定最终书名
  - 必要时与编辑讨论确定

### 3. 内容提要撰写
- **提要要素**：
  - 核心情节概括（不剧透结局）
  - 主要人物介绍
  - 题材类型说明
  - 作品特色和卖点
- **提要要求**：
  - 简洁精炼（200-500字）
  - 吸引读者兴趣
  - 准确反映作品内容
  - 突出作品特色
- **提要用途**：
  - 封底介绍
  - 网络平台简介
  - 宣稿材料
  - 宣传文案基础

### 4. 作者简介撰写
- **基本信息**：
  - 笔名或真名
  - 出生年份和籍贯
  - 职业 or 身份
  - 现居住地
- **创作经历**：
  - 创作年限
  - 已发表或出版作品
  - 获奖情况
  - 创作风格或理念
- **简介风格**：
  - 简洁大方（100-300字）
  - 符合作者形象定位
  - 适度展示但不夸张
  - 可根据不同场合调整

### 5. 出版材料准备
- **投稿材料**（传统出版）：
  - 完整书稿
  - 内容提要
  - 作者简介
  - 章节大纲或故事梗概
  - 目标读者分析
  - 市场定位说明
  - 同类作品分析
  - 作者联系方式
- **投稿信撰写**：
  - 简洁的自我介绍
  - 作品的基本信息
  - 作品的特色和卖点
  - 为什么选择该出版社
  - 联系方式
- **自出版材料**：
  - 书稿最终版
  - 封面设计要求或方案
  - 内页排版要求
  - 书号申请材料
  - 印刷要求说明

### 6. 版权信息准备
- **版权声明**：
  - 版权所有声明
  - 侵权必究声明
  - 版权归属说明
- **ISBN申请**：
  - 了解申请流程
  - 准备申请材料
  - 等待审批
- **CIP数据**：
  - 申请CIP数据核字号
  - 准备相关材料
  - 配合审核

### 7. 最终版本确认
- **版本冻结**：
  - 确定最终版本
  - 停止修改
  - 冻结版本号
  - 妥善保存
- **备份存档**：
  - 多处备份（本地、云端、移动硬盘）
  - 不同格式保存（Word、PDF、纯文本）
  - 版本标注清晰
  - 定期检查备份完整性
- **版本说明**：
  - 版本号和日期
  - 与前版本的差异说明
  - 最终确认的修改内容
  - 冻结确认记录

### 8. 出版路径确认
- **传统出版**：
  - 确定投稿出版社名单
  - 了解各出版社特点
  - 准备投稿材料
  - 制定投稿计划
- **网络平台**：
  - 选择合适的平台
  - 了解平台规则
  - 准备连载材料
  - 制定更新计划
- **自费出版**：
  - 选择出版公司
  - 了解费用和服务
  - 准备相关材料
  - 签订合同

## 评估标准

### 定量指标
- **文字错误率**：≤ 0.05%（万分之零点五）
- **材料完整度**：所有出版材料准备齐全
- **备份安全性**：至少3处备份
- **格式规范性**：格式错误率为0

### 定性指标
- **书名恰当**：书名符合作品，有吸引力
- **提要准确**：内容提要准确反映作品
- **简介得体**：作者简介恰当展示作者
- **材料专业**：出版材料专业规范
- **版本确定**：最终版本确认无误

### 能力等级
- **初级**：能完成基本定稿，但出版材料不够专业
- **中级**：定稿完整，出版材料规范，达到出版要求
- **高级**：定稿优秀，出版材料专业，有市场吸引力
- **专家级**：定稿完美，出版材料出色，具备畅销潜力

## 实践建议

1. **最后把关**：定稿前最后一次全面检查
2. **书名重视**：书名影响深远，慎重确定
3. **材料专业**：出版材料体现专业素养
4. **备份多重**：多处备份，确保安全
5. **版本冻结**：定稿后不再修改，保持稳定
6. **路径明确**：明确出版路径，做好准备

## 输出成果

- 最终定稿（冻结的最终版本）
- 书名确定（正式书名）
- 内容提要（用于封底和宣传）
- 作者简介（用于书籍和宣传）
- 出版材料包（投稿或出版的所有材料）
- 版本说明（最终版本的详细说明）
- 备份记录（备份位置和管理方案）
- 出版计划（后续出版路径和计划）
