# 三稿精修技能

## 核心能力描述

三稿精修是小说创作的精细化打磨阶段，该技能帮助创作者润色语言文字、精炼表达方式、统一风格基调、强化细节描写，使作品在文字层面达到出版级水准。

## 关键操作步骤

### 1. 语言润色
- **词汇选择**：
  - 避免重复用词
  - 选择精准生动的词汇
  - 根据语境调整用词
  - 避免陈词滥调和套话
- **句式优化**：
  - 句式长短搭配，避免单调
  - 调整句式节奏
  - 避免冗长复杂的句子
  - 增强句子的表现力
- **修辞运用**：
  - 比喻、拟人等修辞的恰当使用
  - 避免过度修辞
  - 修辞服务于表达，不炫技
  - 创造新颖独特的修辞

### 2. 描写精炼
- **细节选择**：
  - 删除无意义的细节
  - 强化有表现力的细节
  - 选择独特新颖的细节
  - 细节服务于人物和情节
- **描写方式**：
  - 多用展示，少用说明
  - 调动五感描写
  - 具体生动，避免抽象
  - 详略得当
- **环境描写**：
  - 删除冗长的环境描写
  - 强化氛围渲染的描写
  - 环境描写与情节人物结合
  - 避免为描写而描写

### 3. 对话打磨
- **对话精炼**：
  - 删除无意义的对话
  - 精简冗长的对话
  - 强化对话的潜台词
  - 增加对话的张力和趣味
- **对话提示语**：
  - 减少"说"的重复使用
  - 用动作代替提示语
  - 提示语与对话配合
  - 避免过度解释对话含义
- **对话个性化**：
  - 强化各人物的语言风格
  - 对话符合人物身份和性格
  - 增加人物的口头禅或特色表达
  - 区分不同人物的说话方式

### 4. 叙事节奏调整
- **段落节奏**：
  - 调整段落长度，避免单调
  - 动作场景用短段落
  - 抒情场景可用长段落
  - 段落间衔接自然
- **句子节奏**：
  - 长短句搭配
  - 快节奏场景用短句
  - 慢节奏场景用长句
  - 节奏服务于氛围
- **信息密度**：
  - 控制每段的信息量
  - 重要信息突出
  - 避免信息过载
  - 留白与充实平衡

### 5. 风格统一
- **基调一致**：
  - 整体风格基调统一
  - 避免风格跳跃
  - 根据题材确定风格
  - 风格服务于内容
- **叙事视角统一**：
  - 视角转换自然明确
  - 避免视角混乱
  - 视角选择服务于表达
  - 保持视角的一致性
- **语言风格统一**：
  - 避免古今语言混杂
  - 避免书面语口语语不当混用
  - 保持叙述语言的一致性
  - 人物语言与叙述语言区分

### 6. 细节强化
- **感官描写**：
  - 增加视觉、听觉、嗅觉、触觉、味觉描写
  - 感官描写具体生动
  - 避免抽象的感官描写
- **情感细节**：
  - 用具体细节表现情感
  - 避免直接陈述情感
  - 情感细节真实可信
  - 情感表达克制有力
- **象征意象**：
  - 强化反复出现的意象
  - 意象的象征意义明确
  - 意象贯穿始终
  - 避免过度象征

### 7. 文字精简
- **删除冗余**：
  - 删除无意义的修饰语
  - 精简重复表达
  - 删除过度解释
  - 删除无关细节
- **合并优化**：
  - 合并表达相同内容的段落
  - 用更精炼的表达替代冗长段落
  - 一句话能说清的不用两句
  - 删繁就简

### 8. 错误修正
- **文字错误**：
  - 错别字检查
  - 标点符号规范
  - 语法错误修正
  - 用词不当修正
- **格式规范**：
  - 段落格式统一
  - 对话格式规范
  - 引号、书名号等标点规范
  - 数字、日期格式统一

### 9. 朗读检验
- **朗读测试**：
  - 朗读全文，检验语感
  - 发现拗口之处
  - 检查节奏是否流畅
  - 检查对话是否自然
- **听觉反馈**：
  - 根据朗读感受调整
  - 修正不流畅的句子
  - 调整节奏和韵律
  - 强化语言的音乐性

## 评估标准

### 定量指标
- **文字错误率**：错别字、标点错误率 ≤ 0.1%
- **重复用词**：同一段落重复用词 ≤ 3次
- **冗余删除**：删除冗余文字占比 10-20%
- **格式规范**：格式错误率 ≤ 0.1%

### 定性指标
- **语言精炼**：表达简洁有力，无冗余
- **生动形象**：描写具体生动，有画面感
- **风格统一**：整体风格协调一致
- **节奏流畅**：阅读体验流畅舒适
- **情感真挚**：情感表达真实动人
- **细节有力**：细节选择精准，表现力强

### 能力等级
- **初级**：能修正明显错误，语言基本通顺
- **中级**：语言流畅，风格统一，有一定文采
- **高级**：语言精炼生动，风格鲜明，细节有力
- **专家级**：语言达到文学性高度，风格独特，文字优美

## 实践建议

1. **朗读检验**：多朗读，用耳朵发现问题
2. **慢工细活**：精修需要耐心，逐段打磨
3. **读者视角**：考虑读者的阅读体验
4. **适度留白**：不过度修饰，保持自然
5. **风格自觉**：明确自己的风格追求
6. **学习借鉴**：学习优秀作品的语言特点

## 约束条件

### 阶段门控

**三稿精修完成后，必须通过以下门控检查才能进入下一阶段**：

1. **语言检查**：语言是否精炼生动？冗余是否已删除？
2. **风格检查**：整体风格是否统一？
3. **对话检查**：对话是否自然？是否符合人物语言风格？
4. **错误检查**：文字错误率是否达标（≤ 0.1%）？
5. **朗读检查**：是否通过朗读检验？语感是否流畅？

**门控不通过则**：继续精修，直到通过门控检查。
