# 世界观构建技能

## 核心能力描述

世界观构建是小说创作的基础工程，该技能帮助创作者建立故事发生的时空背景、社会规则、文化体系，为人物活动和情节展开提供可信的舞台和逻辑支撑。

## 关键操作步骤

### 1. 时代背景设定
- **历史时代选择**：
  - 古代（具体朝代或架空古代）
  - 近代（民国、建国初期等）
  - 现代（当代社会）
  - 未来（近未来或远未来）
- **时代特征**：
  - 政治制度和权力结构
  - 经济形态和阶级分化
  - 科技水平和生活方式
  - 思想观念和价值体系
- **时代氛围**：
  - 社会风气和精神面貌
  - 时代特有的矛盾和问题
  - 时代发展的趋势和方向

### 2. 地理环境构建
- **宏观地理**：
  - 世界地图或区域地图
  - 地形地貌（山川、河流、平原）
  - 气候特点
  - 自然资源分布
- **微观地理**：
  - 故事主要发生地点
  - 城市布局和建筑风格
  - 重要场所设计（宫殿、市场、学校等）
  - 地方特色和风物
- **地理与故事**：
  - 地理环境对情节的影响
  - 地理环境对人物性格的塑造
  - 地理环境象征意义

### 3. 社会规则体系
- **政治规则**：
  - 政治制度和权力运作
  - 法律体系和司法制度
  - 官僚机构和等级制度
  - 政治派系和权力斗争
- **经济规则**：
  - 经济制度和生产方式
  - 货币体系和物价水平
  - 职业分工和收入差距
  - 商业运作和市场规则
- **社会规则**：
  - 社会阶层和流动机制
  - 礼仪规范和社交规则
  - 婚姻家庭制度
  - 教育选拔制度
- **潜规则**：
  - 官场/职场潜规则
  - 江湖规矩
  - 行业惯例
  - 不成文的禁忌

### 4. 文化体系构建
- **价值观念**：
  - 主流价值观
  - 不同群体的价值差异
  - 价值冲突和演变
- **宗教信仰**：
  - 宗教体系和神祇设定
  - 宗教仪式和禁忌
  - 宗教组织和社会影响
  - 信仰冲突
- **文化传统**：
  - 节日习俗
  - 人生礼仪（出生、成年、婚嫁、丧葬）
  - 民间传说和神话
  - 文化禁忌
- **艺术审美**：
  - 文学艺术特点
  - 建筑服饰风格
  - 审美标准和趣味

### 5. 特殊体系设计（根据题材需要）

#### 奇幻/仙侠设定
- **魔法/修真体系**：
  - 能量来源和运行原理
  - 等级划分和晋升机制
  - 修炼方法和资源
  - 能力的限制和代价
- **种族设定**：
  - 不同种族的特点
  - 种族间关系
  - 种族文化和冲突
- **神秘存在**：
  - 神明/仙魔设定
  - 灵异生物
  - 神器法宝

#### 科幻设定
- **科技体系**：
  - 核心科技原理
  - 科技发展水平
  - 科技的社会影响
  - 科技的限制和风险
- **未来社会**：
  - 社会组织形式
  - 人机关系
  - 新的社会问题
- **宇宙设定**：
  - 星际政治
  - 外星文明
  - 宇宙规则

#### 历史架空设定
- **历史改编原则**：
  - 保留的历史要素
  - 改变的设定
  - 改编的合理性
- **虚构王朝**：
  - 朝代更替
  - 帝王世系
  - 重大历史事件

### 6. 世界观一致性维护
- **内部逻辑自洽**：各设定之间不矛盾
- **因果链条完整**：现象有合理成因
- **细节相互支撑**：细节呼应整体设定
- **发展演变合理**：世界随时间演变

### 7. 世界观呈现策略
- **渐进式呈现**：随情节发展逐步展开
- **人物视角**：通过人物认知呈现世界
- **冲突展现**：通过矛盾冲突揭示规则
- **细节暗示**：用具体细节暗示宏大设定
- **避免说明文**：融入情节和人物，避免大段说明

## 所需知识储备

### 历史知识
- 中国历史各朝代特点
- 世界历史重要时期
- 历史演变规律
- 历史细节考证

### 社会科学
- 政治学基本原理
- 经济学基础知识
- 社会学理论
- 人类学和文化研究

### 自然科学（科幻需要）
- 物理学前沿理论
- 生物学和进化论
- 天文学和宇宙学
- 科技发展趋势

### 神话与民俗
- 中外神话体系
- 民间传说和习俗
- 宗教文化知识
- 神秘学理论

### 工具使用

- 地图绘制工具
- 时间线管理
- 设定数据库

## 评估标准

### 定量指标
- **设定完整度**：核心设定覆盖率 ≥ 80%
- **细节丰富度**：每个主要设定有3个以上细节支撑
- **一致性检验**：设定间无矛盾冲突
- **呈现比例**：世界观说明文字占比 ≤ 5%

### 定性指标
- **逻辑自洽**：世界观内部逻辑严密
- **真实可信**：设定具有说服力和可信度
- **独特性**：世界观有独特创意和辨识度
- **深度**：世界观有历史纵深和文化厚度
- **功能性**：世界观有效支撑情节和人物
- **呈现自然**：世界观融入故事，不突兀

### 能力等级
- **初级**：能建立基本世界观，但较简单粗糙
- **中级**：能构建完整世界观，逻辑自洽
- **高级**：世界观复杂精妙，细节丰富，深度呈现
- **专家级**：创造独特世界观体系，具有开创性和影响力

## 实践建议

1. **深入研究**：对相关领域进行深入研究，确保专业性
2. **适度设定**：设定服务于故事，避免过度设定
3. **留有余地**：不必事事设定，给创作留空间
4. **持续完善**：写作中不断补充和完善设定
5. **读者视角**：考虑读者理解和接受程度
6. **建立档案**：系统记录所有设定，便于查阅

## 约束条件

### 架空化处理流程

**当世界观涉及历史架空设定时，必须按以下流程进行架空化处理**：

1. **确定架空化范围**：
   - 朝代名称
   - 地名
   - 人名（帝王、名臣等）
   - 机构名称
   - 年号
   - 重大历史事件

2. **设计架空名称**：
   - 至少提供5个备选名称
   - 每个名称需详细说明构思依据、文化内涵及与世界观设定的契合点
   - 名称需考虑历史背景、文化特征、社会结构、地理环境及核心主题

3. **选定最终名称**：
   - 与用户讨论确定最终架空名称
   - 记录选择理由

4. **更新所有正式文档**：
   - 大纲文档
   - 世界观文档
   - 人物小传
   - 正文

5. **保留参照物文档**：
   - 灵感捕捉文档中的参照物保留不动
   - 市场调研文档中的真实名称保留不动

6. **建立架空名称对照表**：
   - 真实名称 → 架空名称
   - 便于后续写作时查阅

### 变更传播机制

**世界观构建阶段发生关键变更时，必须触发变更传播检查**：

1. **识别变更影响范围**：哪些设定发生了变化？
2. **列出受影响文档**：大纲、人物小传等文档是否需要同步更新？
3. **逐一更新**：按清单更新所有受影响的文档
4. **验证一致性**：确认更新后的跨阶段一致性

**变更分级**：
- **S级变更**（核心设定变更）：全链路更新，如朝代名称、能力体系根本性变更
- **A级变更**（重要设定调整）：更新直接关联文档，如社会规则调整
- **B级变更**（细节修改）：只更新当前文档，如地名修改

**影响范围识别清单**（按层级排查）：
- 第一层：直接关联（能力体系 → 社会规则 → 历史设定 → 地理环境）
- 第二层：人物关联（主角小传 → 反派小传 → 配角小传 → 人物关系 → 成长弧线）
- 第三层：大纲关联（章节大纲 → 冲突地图 → 关键场景 → 伏笔设计）
- 第四层：写作关联（语言风格 → 自检清单 → 对话示例）

**更新顺序规则**：
1. 先更新世界观自身（能力体系 → 社会规则 → 历史 → 地理 → 文化）
2. 再更新受影响的人物设定
3. 再更新受影响的大纲
4. 最后更新写作指南

**变更记录模板**：
```
变更编号：CP-WB-[序号]
变更日期：YYYY-MM-DD
触发阶段：世界观构建
变更内容：[具体描述]
变更原因：[为什么需要变更]
变更级别：S/A/B级
受影响文档：[列表]
更新状态：进行中/已完成
验证结果：通过/未通过
```

**变更回滚机制**：
- 变更前备份当前版本
- 变更过程中如发现方向不对，可回滚到变更前版本
- 变更完成后验证不通过，也可回滚

**关系链断裂检测**（S/A级变更必须执行）：
1. 新设定是否与现有角色身份冲突？（每个角色的身份/职业/知识是否与新设定矛盾）
2. 新设定是否与现有角色关系冲突？（角色之间的信息差、互动是否与新设定矛盾）
3. 新设定是否与现有情节冲突？（已发生的情节、角色的行为是否与新设定矛盾）

**修复方案评估**（发现断裂点后）：
1. 逻辑自洽性：修复方案是否与现有设定逻辑自洽？
2. 戏剧张力：修复方案是否增加了戏剧张力？
3. 角色深度：修复方案是否深化了角色？
4. 过审风险：修复方案是否增加过审风险？
5. 修改成本：修复方案需要修改多少文档？

**断裂点依赖分析**（多个断裂点时）：
1. 列出所有断裂点
2. 分析断裂点之间的依赖关系
3. 确定修复顺序（先修复上游断裂点）
4. 上游修复方案确定后，再评估下游断裂点

**更新顺序策略选择**：
- 顺向传播（世界观→人物→大纲→写作）：变更源头在上游，按当前规则"先更新自身→再更新下游"执行
- 逆向传播（大纲→人物→世界观）：变更源头在下游，需先评估底层能否支撑变更
  - 底层可以支撑：按当前规则执行
  - 底层无法支撑：先修复底层，再更新当前阶段

**变更可行性评估**（S级变更必须执行，在更新之前）：
1. 变更是否与核心主题冲突？（如果冲突，是否可以调和？）
2. 变更是否需要重写超过50%的文档？（如果是，考虑是否值得）
3. 变更是否改变故事的根本性质？（如果是，需要重新评估）
4. 变更的修复成本是否远大于收益？（如果是，考虑替代方案）
评估结果：可行 / 有条件可行 / 不可行（否决变更，寻找替代方案）

**回溯修改规则**（变更需要修改已完成阶段的文档时）：
1. 标记回溯原因和范围
2. 修改完成后重新执行该阶段的门控检查
3. 确认修改不会导致该阶段其他文档不一致
4. 只修改受影响的文档，不做"顺便"修改
5. 修改后必须更新变更记录
6. 从回溯阶段开始，逐阶段检查是否有新的不一致

**常见触发场景**：
- 朝代名称变更
- 能力体系调整
- 社会规则重大修改
- 地理环境重大调整

### 阶段门控

**世界观构建完成后，必须通过以下门控检查才能进入下一阶段**：

1. **完整性检查**：时代背景、地理环境、社会规则、文化体系是否完整？
2. **一致性检查**：世界观内部是否逻辑自洽？是否与大纲一致？
3. **过审检查**：世界观是否遵守过审约束？是否使用了架空表述？
4. **功能性检查**：世界观是否为人物活动和情节展开提供了充分支撑？
5. **架空化检查**：如涉及历史架空，架空化处理是否完成？对照表是否建立？

**门控不通过则**：修正世界观设定，直到通过门控检查。

## 输出成果

- 世界观总纲（整体设定概述）
- 地理环境设定（地图和地点描述）
- 社会规则手册（政治、经济、社会规则）
- 文化体系文档（价值、宗教、习俗）
- 特殊体系设定（魔法、科技等特殊规则）
- 世界观呈现计划（如何融入故事呈现）
- 设定查询手册（便于写作时查阅）
- 架空名称对照表（真实名称与架空名称的对照，如适用）
