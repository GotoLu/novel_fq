## 🌍 世界观构建

### 世界类型
**现代都市侦探世界**

### 时空背景
- **时代**：现代中国
- **地点**：虚构城市"江城市"
- **氛围**：轻松幽默的现代都市

### 核心设定

#### 侦探体系
```
侦探等级体系：
├── 警方侦探（正式编制）
│   ├── 刑警队
│   └── 派出所
├── 私家侦探（持证营业）
│   ├── 正规侦探社
│   └── 个人侦探
└── 民间协查员（编外人员）
    └── 李大嘴属于此类
```

#### 案件类型
- **民事案件**：寻物、调查背景等
- **刑事案件**：盗窃、诈骗、杀人等
- **疑难案件**：常规方法难以解决的案件

#### 破案规则
1. **证据为王**：需要确凿证据
2. **程序合法**：警方需要合法程序
3. **民间协查**：协查员可以灵活调查

### 社会结构

#### 主要势力
| 势力 | 代表人物 | 特点 |
|------|---------|------|
| 警方 | 张所长、王小美 | 正规、严谨 |
| 私家侦探 | 赵精英 | 专业、精英 |
| 民间协查 | 李大嘴 | 非正规、独特 |
| 社区力量 | 刘大妈 | 热心、消息灵通 |

### 环境设定

#### 重要地点
1. **幸福派出所**：李大嘴的工作地点
2. **李大嘴的出租屋**：简陋但温馨
3. **阳光社区**：刘大妈的社区，案件多发地
4. **江城大酒店**：高档场所，案件发生地
5. **老街巷子**：充满烟火气的地方

## 📖 章节大纲

### 开篇：歪打正着的侦探（第1-3章）

#### 第一章：大嘴侦探登场
**核心事件**：李大嘴出场，展示"反向推理"能力  
**字数**：3000字

**情节要点**：
- 李大嘴在派出所的日常
- 刘大妈来报案：广场舞音响丢了
- 李大嘴的推理："音响丢了→音响不想跳舞了→音响有思想→思想从哪来？→电→插座→插座在哪？→墙角→果然！"
- 结果：在墙角找到音响（被小偷藏的）
- 王小美吐槽：推理完全错误，但结果正确

**幽默点**：
- 李大嘴一本正经的推理
- 王小美的吐槽
- 张所长的无奈笑容

**悬念**：李大嘴到底是什么来头？

#### 案件设定
**案件类型**：盗窃案  
**难度**：中等  
**核心谜题**：传家宝在密室中消失

#### 章节规划

| 章节 | 标题 | 核心事件 | 字数 |
|------|------|---------|------|
| 第4章 | 密室失窃 | 传家宝在密室消失 | 3000 |
| 第5章 | 不可能的犯罪 | 分析密室，看似不可能 | 3000 |
| 第6章 | 调查开始 | 李大嘴开始调查 | 3000 |
| 第7章 | 歪理推理 | 李大嘴的独特推理 | 3000 |
| 第8章 | 线索出现 | 发现关键线索 | 3000 |
| 第9章 | 真相大白 | 揭开真相 | 3000 |
| 第10章 | 歪打正着 | 推理错误结果正确 | 3000 |

#### 推理设计

**常规推理路径**（赵精英）：
```
密室失窃 → 凶手有钥匙 → 谁有钥匙？ → 主人、管家、儿子 → 排除法 → 儿子是凶手
```

**李大嘴推理路径**：
```
密室失窃 → 密室很神秘 → 神秘的东西吸引人 → 谁被吸引？ → 主人的情敌 → 情敌是谁？ → 邻居老王 → 老王是凶手
```

**真相**：
- 凶手确实是邻居老王
- 但推理完全错误
- 老王是通过通风管道进入的（李大嘴没推理出来）

## 📊 逻辑推理设计

### 推理规则

1. **每个案件都有合理的真相**
2. **李大嘴的推理虽然错误，但指向正确方向**
3. **常规推理有时会陷入误区**
4. **李大嘴的"反向推理"能看到盲区**

### 推理模式

```
常规推理：
观察 → 分析 → 推理 → 验证 → 结论

李大嘴推理：
观察 → 误解 → 歪理 → 意外 → 正确结论
```
