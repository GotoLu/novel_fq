# 第一章 大嘴侦探登场（初稿v1）

幸福社区活动室里，一群大妈围着一个人。

"李侦探，您可得帮我们看看啊！"

被围在中间的是个瘦高个子的年轻人，穿着大两号的灰色西装，袖子长得盖住半个手掌；鼻梁上架着圆框眼镜，镜片厚得像啤酒瓶底；头发乱得像刚从被窝钻出来，却又梳了个中分。

他叫李大嘴，自称"民间侦探"。

此刻，他正一脸严肃地摸着下巴，摆出福尔摩斯思考问题的姿势。

"各位大妈，请安静。"他抬起手，声音沉稳有力，"让我来分析一下这个案件。"

大妈们立刻安静下来，用崇拜的眼神看着他。

"事情是这样的，"刘大妈——广场舞领队，也是李大嘴的忠实粉丝——开口道，"我们广场舞队的音响，昨天晚上明明放在活动室里，今天早上来就不见了！活动室的门窗都锁着，钥匙只有我有，我发誓我没拿！这音响价值三千多呢，是我们凑钱买的，您可得帮我们找回来啊！"

"密室消失案？"李大嘴的眼睛亮了，"有意思，非常有意思。"

他开始在活动室里转圈，一边转一边观察。他的观察角度很奇怪——不看门锁，不看窗户，反而盯着墙角的垃圾桶看了半天，又蹲下来研究地板上的灰尘，甚至还凑到空调出风口闻了闻。

大妈们面面相觑，不知道他在干什么。

"李侦探，您在找什么？"刘大妈小心翼翼地问。

"线索。"李大嘴头也不抬地说，"真相往往藏在最不起眼的地方。"

他又转了几圈，突然停下脚步，目光落在活动室角落的一个旧纸箱上。

"找到了！"他大步走过去，掀开纸箱——里面空空如也。

"呃……"李大嘴愣了一下，但立刻恢复自信，"这个纸箱虽然空的，但它告诉了我一个重要信息。"

"什么信息？"大妈们齐声问。

"这个纸箱……"李大嘴一本正经地说，"曾经装过东西。"

大妈们："……"

"而且，"李大嘴继续推理，"装的东西，很可能就是音响。"

"为什么？"刘大妈问。

"因为纸箱的大小，正好能装下音响。"李大嘴振振有词，"这不是巧合，这是命运的安排！"

大妈们虽然觉得这推理有点奇怪，但李大嘴说得如此笃定，她们又不敢质疑。

"那么，音响去哪了呢？"刘大妈追问。

李大嘴又开始在活动室里转圈。这次他盯着天花板看了半天，又去敲了敲墙壁，最后走到窗边，看着窗外的广场。

窗外，几个老头正在下象棋。

李大嘴的眼睛眯了起来。

"根据我的推理，"他缓缓开口，声音里透着自信，"这个案件的关键，在于理解音响的'心理'。"

"音响还有心理？"一个大妈忍不住问。

"万物皆有灵。"李大嘴一本正经地说，"音响每天陪你们跳广场舞，听的都是《最炫民族风》《小苹果》这些歌。你们想想，它会不会听腻了？"

大妈们面面相觑。

"所以，"李大嘴继续推理，"音响可能想换换口味，想听听别的歌。比如……"他看向窗外下象棋的老头们，"比如象棋的落子声。"

"您的意思是……"刘大妈眼睛瞪大。

"我的推理是，"李大嘴摆出结论性的手势，"音响自己跑到下象棋的地方去了！"

大妈们彻底懵了。

"李侦探，音响是死的，它怎么可能自己跑？"

"这就是这个案件的神秘之处。"李大嘴神秘地说，"但我相信，只要我们顺着音响的'心理'去找，一定能找到它。"

说完，他大步走出活动室，直奔广场上的象棋摊。

大妈们半信半疑地跟在后面。

李大嘴走到象棋摊前，看着几个老头下棋。

"大爷们，"他问，"你们这儿有没有看到一个音响？"

"音响？"一个老头抬起头，"没有啊。"

李大嘴皱起眉头，"奇怪，根据我的推理，音响应该在这里。"

他开始在象棋摊周围转悠，又去旁边的花坛看了看，还掀开了几张报纸。

什么都没有。

大妈们开始窃窃私语："李侦探这次是不是推理错了？"

"不会吧，他以前破案都很准的。"

"可是音响怎么可能自己跑呢？"

李大嘴听到了这些议论，但他丝毫不慌。他坚信自己的推理是正确的——虽然他也说不清为什么坚信。

他又转了一圈，目光突然落在象棋摊旁边的一个手推车上。

那是一辆卖早点的手推车，摊主正在收拾东西准备收摊。

李大嘴走过去，"老板，你这车里装的什么？"

"早点啊，包子油条豆浆。"摊主奇怪地看着他。

"能让我看看吗？"

摊主虽然觉得莫名其妙，但还是掀开了盖子。

里面果然是早点，没有音响。

李大嘴又问："你今天早上几点出摊的？"

"五点半。"

"五点半……"李大嘴若有所思，"那个时候，广场上有人吗？"

"有啊，这帮老头，天天五点就来占座下棋。"

李大嘴眼睛一亮，"那他们下棋的时候，有没有听到什么奇怪的声音？"

"奇怪的声音？"摊主想了想，"好像……是有过一阵音乐声，但很快就没了。"

"音乐声！"李大嘴猛地一拍手，"果然！"

他转向大妈们，"你们听到了吗？音乐声！这说明什么？"

"说明……音响在这里响过？"

"不，"李大嘴摇了摇头，"说明音响确实来过这里，但后来又离开了。"

"那它去哪了？"

李大嘴又开始思考。他看着广场，看着周围的环境，突然，他的目光锁定在广场边缘的一排储物柜上。

那是社区给居民存放运动器材的柜子，每个柜子都有锁。

"那里！"李大嘴指着储物柜，"音响一定在那里！"

他快步走过去，大妈们赶紧跟上。

储物柜有十几个，都锁着。李大嘴一个个查看，最后停在一个柜子前。

"这个。"他笃定地说，"音响就在这个柜子里。"

"你怎么知道？"刘大妈问。

"因为，"李大嘴一本正经地说，"这个柜子的锁，和其他柜子的锁不一样。"

大妈们仔细看了看，发现这个柜子的锁确实有点不一样——锁孔上有新鲜的划痕。

"这说明什么？"

"说明有人最近开过这个锁。"李大嘴说，"而且开得很急，所以留下了划痕。"

"那我们现在怎么办？叫开锁师傅？"

"不用，"李大嘴掏出一根铁丝，"让我来。"

他把铁丝插进锁孔，摆弄了几下，"咔哒"一声，锁开了。

大妈们惊呆了："李侦探还会开锁？"

"侦探的基本技能。"李大嘴得意地说。

他拉开柜门——里面果然是那台音响！

"找到了！"大妈们欢呼起来。

刘大妈激动得握住李大嘴的手，"李侦探，您太厉害了！可是，您是怎么知道音响在这里的？"

李大嘴清了清嗓子，准备解释他的推理过程。

"根据我的推理，"他一本正经地说，"音响想听象棋的落子声，所以它跑到象棋摊那边。但是象棋摊太吵了，音响觉得不够优雅，所以它又离开了。它想找个安静的地方，就看到了这些储物柜。储物柜既安静又安全，所以音响决定躲进去休息。"

大妈们听得一愣一愣的。

"可是……"一个大妈忍不住问，"音响是死的，它怎么可能自己跑来跑去？"

"这就是这个案件的真相，"李大嘴神秘地说，"表面上看，音响是自己跑的。但实际上——是有人把它搬走的。"

"谁？"

"这个人，"李大嘴看向象棋摊那边的老头们，"就是每天最早来占座下棋的人。他为了占座，凌晨五点就来了。但他发现象棋摊的位置被音响占了——因为你们每天早上也在广场跳舞。"

大妈们恍然大悟："对！我们每天六点开始跳舞，音响五点半就搬出来！"

"所以，"李大嘴继续推理，"这个老头为了占座，趁活动室没人的时候，偷了音响。他先把音响藏在早点车里——所以摊主听到了音乐声——然后等你们来了，他又把音响转移到储物柜里。"

大妈们听得目瞪口呆。

"李侦探，您是怎么推理出来的？这太厉害了！"

李大嘴微微一笑，摆出谦虚的姿势，"这只是基本操作。真相只有一个，但通往真相的路有很多条。而我走的，恰好是最弯的那条。"

大妈们更加崇拜了。

但就在这时，刘大妈突然想起什么，"等等，李侦探，您刚才不是说音响是自己跑的吗？怎么又变成老头偷的了？"

李大嘴愣了一下，"呃……"

"而且，"刘大妈继续追问，"您推理说音响想听象棋声，这个……音响真的有心理吗？"

李大嘴咳嗽了一声，"这个嘛……其实我的推理过程是——"

他停顿了一下，试图组织语言。

实际上，他的推理过程是这样的：

1. 音响在密室消失 → 不可能是凭空消失 → 一定有人拿走了
2. 活动室门窗锁着 → 有钥匙的人只有刘大妈 → 刘大妈没拿 → 还有谁有钥匙？
3. 李大嘴不知道还有谁有钥匙，所以他换了个角度——
4. 音响消失的时间是昨晚到今早 → 这段时间谁会来活动室？
5. 李大嘴不知道，所以他又换了个角度——
6. 音响最后会在哪里？→ 广场 → 为什么在广场？→ 因为广场是活动室外面
7. 音响在广场的哪里？→ 象棋摊 → 为什么在象棋摊？→ 因为象棋摊离活动室最近
8. 但象棋摊没有音响 → 音响去哪了？→ 储物柜 → 为什么在储物柜？→ 因为储物柜能藏东西

整个推理过程，逻辑混乱，跳跃极大，充满了"因为所以"但实际上毫无因果关系的推导。

但神奇的是——结果是对的。音响确实在储物柜里。

这就是李大嘴的"歪打正着"体质。

"其实，"李大嘴终于开口，"我的推理方法是——'大嘴推理法'。"

"大嘴推理法？"大妈们好奇地问。

"对，"李大嘴一本正经地说，"这是一种非常先进的推理方法，核心思想是：先假设一个结论，然后寻找支持这个结论的证据。如果找不到证据，就换一个结论，继续找。直到找到证据为止。"

大妈们听得云里雾里，但觉得很厉害的样子。

"那您一开始假设的结论是什么？"

"音响在象棋摊。"李大嘴坦然地说。

"但音响不在象棋摊啊。"

"所以我换了一个结论：音响在储物柜。"李大嘴依然坦然，"然后我找到了证据。"

大妈们："……"

这个推理方法，听起来完全是瞎蒙啊！

但李大嘴不这么认为。他觉得，只要结果是正确的，推理过程就不重要。

"总之，"他拍了拍手，"案件破了，音响找到了。刘大妈，您可以把音响拿回去了。"

刘大妈点点头，和几个大妈一起把音响搬了出来。

"李侦探，太感谢您了！"刘大妈再次握住李大嘴的手，"您一定要来我们广场舞队做客，我请您吃最好的早点！"

"不用客气，"李大嘴摆摆手，"为民服务，是我的职责。"

大妈们千恩万谢地走了，李大嘴站在原地，看着她们远去的背影，脸上露出满足的笑容。

虽然他的推理过程完全错误，但他帮助了别人，找到了真相。

这就够了。

"李大嘴，"他对自己说，"你又破了一个案子。看来你的侦探天赋，是真的。"

他整理了一下那件不合身的西装，扶了扶度数不对的眼镜，迈着自信的步伐，朝派出所的方向走去。
