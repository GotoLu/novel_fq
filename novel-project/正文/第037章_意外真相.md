# 第三十七章 意外真相

王小美回来了。

她带回来一份厚厚的调查报告，放在会议室桌上。

"孙磊和马文才的背景调查，"她说，"比预想的复杂。"

赵精英和李大嘴同时看向她。

"先说马文才，"王小美翻开报告，"28岁，计算机专业，鸿运安防技术员。入职两年，月薪6500，租住在开发区一室一厅，未婚，无车。银行流水正常，工资到账后主要用于房租、日常消费和还助学贷款。手机定位记录显示，7月8日凌晨他在家，没有异常活动。"

"排除？"赵精英问。

"基本排除，"王小美说，"他的经济状况不支持八百万的作案动机，手机定位也没有异常。而且他入职才两年，对系统的了解程度有限，不太可能独立完成画面替换和后门设置。"

"孙磊呢？"

王小美翻到下一页，表情变了。

"孙磊，35岁，电子工程专业，鸿运安防项目主管。入职四年，月薪12000，已婚，妻子在一家私立幼儿园当老师。"

她顿了顿。

"但他的银行流水有问题。"

赵精英坐直了。

"什么问题？"

"孙磊的工资卡流水看起来正常，每月工资到账，日常消费合理。但我查了他的另一张卡，一张他妻子名下的储蓄卡。这张卡在过去六个月里，陆续存入了四笔现金，每笔5万，总共20万。"

"现金？"李大嘴问。

"对，现金存入，"王小美说，"没有转账记录，没有来源说明。他妻子月薪4500，不可能存出20万。"

"20万，"赵精英说，"和八百万的古董比起来不多。"

"但这只是目前查到的，"王小美说，"也许还有其他账户，或者有实物交易。而且20万可能只是定金或者销赃的预付款。"

她翻到下一页。

"还有手机定位。7月8日凌晨，孙磊的手机定位显示他在家。但从凌晨2:15到4:20，他的手机关机了。"

"关机？"赵精英眉头一皱。

"关机，"王小美说，"他平时从不关机，我查了他最近三个月的手机记录，只有7月8日凌晨这一次关机。从2:15关机，4:20开机。关机时间刚好覆盖了2:17到3:48的作案时段。"

会议室里安静了几秒。

"2:15关机，4:20开机，"赵精英重复了一遍，"他关机是为了不让手机定位记录他的行踪。"

"但这也说明他不在家，"李大嘴说，"如果他在家睡觉，不需要关机。只有出门，才需要关机。"

"对，"王小美说，"而且4:20开机，说明他4:20才回到家。从他家到文化中心，开车大约25分钟。如果4:20到家，他离开文化中心的时间大约是3:55。"

"3:55，"赵精英看了看笔记本上的时间线，"3:52修改文件，3:55离开。时间吻合。"

---

赵精英站起来，走到白板前。

他在白板上画了一个时间线：

```
2:15  孙磊手机关机
2:17  监控显示瓶子还在（真实画面）
3:00  王刚听到走廊声音（帧间差异开始降低）
3:47  瓶子消失（2号7号摄像头拼接点）
3:48  监控显示瓶子消失（替换画面开始）
3:52  2号7号摄像头文件被修改
3:55  孙磊离开文化中心（推算）
4:20  孙磊手机开机
```

"时间线完全吻合，"赵精英说，"2:15关机出门，3:00左右到达文化中心，通过后门远程访问硬盘录像机，开始替换2号和7号摄像头的画面。3:47进入展厅取走瓶子，同时拼接真实画面和提前录制的画面。3:52修改文件元数据。3:55离开。4:20到家开机。"

他在白板上又画了一个能力线：

```
技术能力：孙磊是项目主管，负责系统设计和调试
→ 他知道硬盘录像机的架构和密码
→ 他有能力设置后门
→ 他有能力替换画面和修改元数据

机会：三个月前的安装调试期间
→ 他有一周时间设置后门
→ 他有权限接触所有设备
→ 调试期间监控中断3-4次，他可以测试后门

动机：妻子名下20万不明现金存入
→ 可能是销赃预付款或定金
→ 需要进一步追查资金来源和古董下落
```

"技术能力、机会、动机，"赵精英说，"三条线都指向孙磊。"

他放下笔，转过身来。

"但还差最关键的一环。"

"什么？"李大嘴问。

"物证，"赵精英说，"时间线和能力分析都是间接证据。我们需要直接证据证明孙磊在7月8日凌晨进入了文化中心，并且操作了硬盘录像机。"

"怎么证明？"

赵精英想了想。

"后门，"他说，"如果孙磊设置了后门，后门的痕迹一定还在硬盘录像机的系统日志里。我需要检查硬盘录像机的系统日志，找到远程访问的记录。"

---

赵精英带着技术处的人去了监控室。

李大嘴跟在后面，虽然他不太懂技术，但他不想错过这一幕。

硬盘录像机放在角落的铁柜里，一台黑色的工业设备，前面板上有几个指示灯，一闪一闪的。

赵精英让技术人员把硬盘录像机的系统日志导出来。

日志很长，几十万条记录，从三个月前安装到现在，每一次启动、关机、录像、回放、设置修改都有记录。

赵精英用搜索功能，输入关键词："remote"。

屏幕上跳出了几条记录：

```
2026-04-12 14:23:17 Remote access enabled - user: admin
2026-04-12 14:23:42 Remote access from 192.168.1.105
2026-04-12 14:25:03 System configuration modified - user: admin
2026-04-12 14:26:15 Remote access disabled
```

"4月12日，"赵精英说，"安装调试期间。有人用admin账户远程访问了硬盘录像机，修改了系统配置，然后关闭了远程访问。"

"admin账户？"李大嘴问，"谁有这个账户？"

"安装调试期间，admin账户由技术人员使用，"赵精英说，"调试完成后，admin密码应该被修改并移交给文化中心的安保主管。但这条记录显示，有人在调试期间开启了远程访问，修改了配置，然后关闭了。"

他继续搜索。

"7月8日。"

屏幕上跳出了更多记录：

```
2026-07-08 02:58:43 Remote access enabled - user: admin
2026-07-08 02:58:51 Remote access from 192.168.1.105
2026-07-08 02:59:07 File modification - CH02, CH07
2026-07-08 03:47:14 File replacement - CH02
2026-07-08 03:47:17 File replacement - CH07
2026-07-08 03:52:17 Metadata modification - CH02, CH07
2026-07-08 03:52:43 Remote access disabled
```

赵精英看着这些记录，深吸一口气。

"7月8日凌晨2:58，有人用admin账户远程访问了硬盘录像机。2:59，开始修改2号和7号摄像头的文件。3:47，替换画面。3:52，修改元数据。3:52，关闭远程访问。"

他指着IP地址。

"192.168.1.105。这是文化中心内网的IP地址。4月12日和7月8日，远程访问来自同一个IP。"

"这个IP对应哪台电脑？"李大嘴问。

赵精英让技术人员查了文化中心的网络设备清单。

"192.168.1.105，"技术人员说，"对应的是2楼走廊侧门旁边的网络接口。"

2楼走廊。侧门旁边。

王刚凌晨3点听到声音的位置。

李大嘴感觉后背一阵发凉。

---

赵精英打了一个电话。

十分钟后，张所长带着两个便衣警察赶到了文化中心。

"证据确凿了？"张所长问。

"系统日志是铁证，"赵精英说，"4月12日调试期间设置后门，7月8日凌晨远程访问替换画面。IP地址指向2楼走廊侧门旁边的网络接口。加上孙磊的手机关机记录和妻子名下的20万不明资金，证据链完整。"

张所长看了赵精英一眼，又看了李大嘴一眼。

"孙磊现在在哪？"

"在公司，"王小美说，"我查了他的手机定位，目前在鸿运安防的办公室。"

张所长点了点头。

"去抓人。"

---

抓捕比想象中顺利。

孙磊坐在鸿运安防的办公室里，面前是一台电脑，屏幕上是一份安防系统的维护合同。两个便衣警察走进去的时候，他抬起头，看了一眼，然后又低下头继续看合同。

"孙磊？"便衣警察问。

"是我，"孙磊说，语气很平静。

"我们是幸福区派出所的，请你配合调查。"

孙磊合上合同，站起来。他没有反抗，没有辩解，甚至没有问一句"调查什么"。

他只是从抽屉里拿出一个U盘，放在桌上。

"你们要找的东西，在这里面，"他说。

便衣警察对视了一眼。

"什么？"

"青花瓷瓶的交易记录，"孙磊说，"买家信息，联系方式，交易地点。都在里面。"

他看着便衣警察，表情平静得像在说今天天气不错。

"我知道你们会来的，"他说，"从你们检查硬盘录像机的那一刻起，我就知道迟早会来。"

---

审讯室里，孙磊很配合。

他交代了一切。

"三个月前安装调试的时候，我就留了后门。admin账户的密码我知道，调试完成后我故意没有把远程访问功能关掉，只是隐藏了。以后我想访问，随时可以。"

"为什么要留后门？"赵精英问。

孙磊沉默了几秒。

"因为我知道，这套系统装完之后，我就是多余的。安装的时候，我是专家，所有人都要听我的。装完了，我就是修电脑的，什么时候坏了什么时候才叫我。"

他看着赵精英。

"你不懂那种感觉。你设计了一个完美的系统，所有人都依赖它，但没有人记得是谁设计的。系统越完美，你越不重要。因为完美的系统不需要维护。"

"所以你让系统变得不完美？"赵精英说。

孙磊苦笑了一下。

"我一开始只是想留个后门，证明我还能控制它。后来有人找到我，问我能不能让监控'看不见'某些东西。我才知道，后门可以换钱。"

"谁找你的？"

"一个古董商人，姓周，"孙磊说，"他说他需要一个'监控看不见'的夜晚。我问他要做什么，他说只是想'借'一件东西。"

"八百万的青花瓷瓶，叫'借'？"

"他原话就是'借'，"孙磊说，"他说会给我50万，事成之后付清。我先收了20万定金。"

"古董现在在哪？"

"不知道，"孙磊说，"我7月8日凌晨取走瓶子，按照约定放在2楼侧门外的消防栓箱里。周姓商人说他会派人取走。之后的事情我不知道。"

"你怎么进入展厅的？"

"侧门，"孙磊说，"调试期间我配了一张万能门禁卡，可以刷开所有门。我2:50左右到达文化中心，从侧门进入2楼，在走廊侧门旁边的网络接口连上笔记本，通过后门远程访问硬盘录像机，开始替换2号和7号摄像头的画面。"

"提前录制的画面是什么时候录的？"

"7月7日晚上，"孙磊说，"文化中心6点闭馆后，我以'系统维护'的名义留了下来。我关掉了展厅的灯光，只留射灯，模拟凌晨的光线环境，然后录制了30分钟的空展厅画面。"

"然后7月8日凌晨，你用这段画面替换了真实的录像？"

"对，"孙磊说，"我先替换了3:00之后的画面，让2号和7号摄像头显示'走廊无人'。然后3:47进入展厅，用万能卡打开展柜电子锁，取出瓶子。3:48离开展厅，在走廊用笔记本完成画面拼接，把3:47之前的真实画面和提前录制的空展厅画面拼在一起。3:52修改了元数据时间戳。"

他说这些的时候，语气平淡，像在描述一个技术方案。

赵精英看着他。

"你有没有想过，你设计的安保系统，本来是保护文物的？"

孙磊沉默了。

"想过，"他说，"但那不重要。重要的是，系统是我设计的，只有我知道它的弱点。如果我不知道，谁也不知道。如果我不利用，这些弱点就永远只是弱点，没有任何价值。"

"弱点不是用来利用的，"赵精英说，"弱点是用来修补的。"

孙磊看着他，嘴角动了动，像是想笑又没笑出来。

"你说得对，"他说，"但太晚了。"

---

李大嘴站在审讯室外面的走廊里，透过单向玻璃看着孙磊。

孙磊坐在审讯椅上，双手放在桌上，手指修长，指甲剪得很整齐。他的表情平静，没有恐惧，没有悔恨，只有一种说不出的疲惫。

"系统越完美，你越不重要。"

李大嘴在心里重复了一遍这句话。

他想到了自己。歪理推理越有效，越没有人理解它为什么有效。他破了一个又一个案子，但每次别人问"你怎么想到的"，他都说不上来。

但赵精英不一样。

赵精英说过："六个假设全部排除，不是因为真相不存在，是因为我们问错了问题。"

问对问题，比找到答案更重要。

而问对问题的那个人，是他。

他翻开歪理笔记，在最后一页写了一行字：

**"歪理找方向，精密做验证。方向比答案重要。"**

他合上笔记本，看着审讯室里的孙磊。

保护安全的人，破坏了安全。设计系统的人，利用了系统。最安全的地方最危险，最可信的证据最可疑。

这就是歪理推理告诉他的：不是不相信任何东西，是不盲目相信任何东西。

包括监控。包括自己。
